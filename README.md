## 🚀 طريقة التثبيت

### الخطوة 1: رفع الملفات
1. قم بفك ضغط ملف `saudi-gold-website.zip`
2. ارفع جميع الملفات إلى المجلد الرئيسي (public_html أو www)

### الخطوة 2: تأكد من الهيكل
```
public_html/
├── index.html          ← الصفحة الرئيسية
├── .htaccess           ← إعدادات السيرفر
├── robots.txt          ← لمحركات البحث
├── sitemap.xml         ← خريطة الموقع
├── css/
│   └── style.css       ← التصميم
├── js/
│   └── app.js          ← الوظائف
├── pages/
│   ├── bars.html       ← صفحة السبائك
│   └── silver.html     ← صفحة الفضة
└── images/             ← الصور (أضف صورك هنا)
```

### الخطوة 3: التحقق
افتح الموقع في المتصفح: `https://saudi-gold.com`

## ⚙️ التخصيص

### تغيير الأسعار
افتح ملف `js/app.js` وعدّل القيم في `CONFIG`:

```javascript
const CONFIG = {
  baseOunceUSD: 2650,    // سعر الأونصة بالدولار
  usdToSar: 3.75,        // سعر الصرف
  // ...
};
```

### ربط API حقيقي
للحصول على أسعار حقيقية، سجّل في:
- [MetalpriceAPI](https://metalpriceapi.com) - الأفضل
- [Gold-API](https://gold-api.com) - مجاني

ثم عدّل دالة `calculatePrices()` في `app.js`

### تغيير الألوان
افتح `css/style.css` وعدّل المتغيرات:

```css
:root {
  --gold-500: #F59E0B;     /* اللون الذهبي الرئيسي */
  --bg-primary: #0A0A0F;   /* لون الخلفية */
  /* ... */
}
```

### إضافة صفحات جديدة
1. انسخ أي صفحة من مجلد `pages/`
2. عدّل المحتوى حسب حاجتك
3. أضف رابط في القائمة
